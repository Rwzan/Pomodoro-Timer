# 🍅 Pomodoro Timer

A simple and clean Pomodoro Study Timer built with HTML, CSS, and JavaScript.

## 🚀 Features
- 25-minute focus timer
- Start / Pause / Reset controls
- Simple UI
- Great for productivity

## 🛠️ Built With
- HTML
- CSS
- JavaScript

## 💡 How it works
The timer counts down from 25 minutes and alerts the user when time is up.

## 📸 Preview
A minimal productivity timer designed for studying and focus sessions.
